# 学生管理系统
##### 要求：
- 用户界面
    - 1.添加学生信息
    - 2.删除学生信息
    - 3.修改学生信息
    - 4.查找学生信息
    - 5.退出系统

-------------------

##### 实现效果大概如下：
>输入数值进入不同功能界面，进行功能操作，逐步实现不同的功能
>- 添加功能：可以添加学生的名字，年龄，班级，学号
>- 删除功能：输入班级和学号，直接删除该学生所有信息
>- 修改功能：输入班级和学号，可以修改该学生的名字和年龄，班级和学号不可修改
>- 查找功能：输入班级和学号，可以查找该学生的名字和年龄
>- 退出系统，关闭终端

-------------------

##### 实现功能方式

- 1.添加功能-用字典students_dict存放，把班级和学号为key，判断是否已经存在，再进行存放学生信息字典new_dict到students_dict中，完成后返回封面
- 2.删除功能-输入班级和学号，建立key，使用get()操作查找字典内容，输出学生信息，询问是否删除，删除成功，返回封面
- 3.修改功能-输入班级和学号，输出学生名字和年龄，询问是否修改，统一不可修改班级和学号，修改名字和年龄，修改成功，返回封面
- 4.查找功能-输出所有学生的信息，使用.items遍历取出students_dict的键值对和学生信息字典，通过f-string输出所有的学生信息内容

-------------------

##### 易错点
>- 双层字典
1.`students_dict` 里面存的是**键值对**对应的学生信息的字典，形如:`{(班级, 学号): {'name': 姓名, 'age': 年龄, 'stu_class': 班级, 'number': 学号}}`
2.`new_dict` 里面存的是学生信息字典，就是`students_dict` 里的**键值对**对应的`{'name': 姓名, 'age': 年龄, 'stu_class': 班级, 'number': 学号}`
